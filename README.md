# API de Cadastro de Usuários

Projeto simples utilizando **Node.js + Express** com arquitetura **MVC**.

## Funcionalidades

- Cadastro de usuários
- Listagem de usuários
- API REST
- Integração Frontend + Backend

## Rodar projeto

Instalar dependências:

npm install

Executar:

npm start

Abrir no navegador:

http://localhost:3000
